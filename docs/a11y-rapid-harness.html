  • 브라우저 200% 확대에서 정보 손실 없이 읽히는가?

How to Use

  1. 이 파일을 브라우저에서 열고 Tab 키로 처음부터 끝까지 이동합니다.
  2. 브라우저 확대를 200%로 올려 레이아웃 깨짐 여부를 확인합니다.
  3. OS 다크모드 전환 후 대비와 포커스 가시성을 다시 점검합니다.
